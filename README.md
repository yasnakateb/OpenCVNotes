# OpenCVNotes

Play with OpenCV in cpp!

## Building on macOS
1. OpenCV can be installed via Homebrew :
   <code>$ brew install opencv</code>
2. Clone the repository.
3. Change the directory to src.

4. Run: 

      <code>$ mkdir build </code> 
      
      <code>$ cd build </code> 
      
      <code>$ cmake .. </code> 
      
      <code>$ make </code>
